## Ordine array
# Inserisci in un array ordinato

Scrivi un programma in C per inserire i valori nell'array (elenco ordinato).
Dati di test:
Inserisci il numero di elementi che desideri inserire (max 100): 5
Inserisci 5 elementi nell'array in ordine crescente:
* elemento - 0 : 2
* elemento - 1 : 3
* elemento - 2 : 4
* elemento - 3 : 7
* elemento - 4 : 8
Inserisci il valore da inserire: 5
L'elenco dell'array esistente è:
2 3 4 7 8
Dopo l'inserimento, l'elenco è:
2 3 4 5 7 8
